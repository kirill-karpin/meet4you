Первая попытка "наладить" инфраструктуру:

если внутри этого каталога выполнить компанду

docker-compose up

то запустится пустая постгрисная база данных с именем SocialNetwork, и к ней можно подключиться с логином паролем:

postgres

postgres

При подключении можно использовать, например, DBeaver:

https://dbeaver.io/download/

Директория Init_Database предназначена для скриптов инициализации базы.
